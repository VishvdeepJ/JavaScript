1. Browser – A software application used to access and view websites on the internet.
  2. Console – A panel in developer tools that displays messages and allows you to run JavaScript directly.
  3. Script Tag – An HTML tag (script) used to add or link JavaScript code in a webpage.
  4. Syntax – The set of rules that define how code must be written for the computer to understand it.
  5. Statement – A single line of code that performs an action.
  6. Variable – A container used to store data values in a program.
  7. Runtime – The time when a program is actually running or being executed.
  8. Compiler / Interpreter – Tools that translate code into a form the computer can understand and execute.
  9. Error Message – A notification that tells you something is wrong in your code.
  10. Developer Tools – Built -in browser tools that help developers inspect, debug, and test web pages.
